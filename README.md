socketio-boilerplate
===========

This is my socket.io application's boilerplate.

It's ready to use:

- socket.io
- express w/ jade, css
- compass(sass w/ animate)
- coffee script
- require.js
- jQuery
- underscore
- Twitter Bootstrap
- Font Awesome
- livereload w/ grunt

Getting Started
-------------

prepare

    $ bundle
    $ bower install
    $ npm install

devel

    $ grunt server

build dist

    $ grunt

check dist

    $ grunt server:dist
